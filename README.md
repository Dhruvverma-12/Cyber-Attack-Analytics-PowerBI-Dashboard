# Project Overview

The **Cyber Attack Analytics & Trends Dashboard** is an advanced **Power BI Business Intelligence solution** designed to analyze global cyber security incidents, financial losses, affected users, attack patterns, industry targets, and security vulnerabilities.

This dashboard transforms raw cyber attack data into **interactive visual insights**, enabling organizations and analysts to identify trends, evaluate risks, and support strategic decision-making.

The project demonstrates expertise in:

- Data Cleaning
- Data Modeling
- DAX Calculations
- Interactive Dashboard Design
- Business Intelligence Reporting
- Data Storytelling

# Dashboard Insights

## Financial Loss by Country

Ranks countries according to total financial loss caused by cyber attacks.

Highlights:

- United Kingdom
- Germany
- Brazil
- Australia
- Japan
- USA
- Russia
- India
- China

---

## Financial Loss by Attack Type

Interactive Donut Chart displaying:

- DDoS
- Phishing
- SQL Injection
- Ransomware
- Malware
- Man-in-the-Middle

allowing quick comparison of attack impact.

---

## Security Vulnerability Analysis

Tree Map visualization identifies major vulnerabilities including:

- Zero-day
- Social Engineering
- Unpatched Systems
- Weak Passwords

making it easier to identify critical security risks.

---

## Industry-wise Analysis

Displays:

- Banking
- Education
- Government
- Healthcare
- IT
- Retail
- Telecom

along with yearly financial losses for each industry.

---

## Attack Source Analysis

Compare attacks originating from:

- Nation-state
- Unknown
- Insider
- Hacker Groups

to understand threat origins.

---

## Yearly Trend Analysis

Dual Line Chart comparing:

- Total Financial Loss
- Total Affected Users

from multiple years, helping identify long-term cybersecurity trends.
